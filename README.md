# JS Playground

JS Playground is an editor that allows you to write and run HTML, CSS, and JavaScript code directly in the browser.

<video src="./screenshoots/video.mp4" controls></video>

## Features

-   Support for HTML, CSS, and JavaScript
-   Powered by the Monaco Editor
-   Built-in console support
-   Create, open, and save projects
-   Export projects as a single HTML file

## Shortcuts

-   You can switch between the HTML, CSS, and JavaScript tabs using the **Ctrl+1**, **Ctrl+2**, and **Ctrl+3** key combinations.
-   You can open a project using the Ctrl+O keys, save the project using the Ctrl+S keys, and export the project as a single HTML document using the Ctrl+E key combination. _(These shortcuts work while you are in the editor.)_
-   Run code instantly using **Ctrl + Enter** inside the Monaco Editor
-   You can also use all built-in Monaco Editor shortcuts

## Purpose

JS Playground is designed for quickly testing and experimenting with HTML, CSS, and JavaScript in a clean and simple environment.
